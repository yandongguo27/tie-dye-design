# 素材处理页面 - 图片矢量化转换

## 📋 功能概述

素材处理页面是一个全新的功能模块，位于纹样设计页面之前，专门用于将彩色图片转换为黑白线稿/矢量图，为后续的纹样设计提供高质量的素材。

## 🎯 核心功能

### 1. 图片上传
- 支持常见图片格式（JPG、PNG、GIF等）
- 拖拽上传或点击选择
- 实时预览上传的图片

### 2. 智能处理流程

#### 步骤1: 边缘检测
- **算法**: Sobel边缘检测
- **功能**: 识别图片中的轮廓和边缘
- **可调参数**: 
  - 边缘检测灵敏度（0-100%）
  - 边缘平滑度（0-5级）

#### 步骤2: 黑白简化
- **功能**: 将彩色图片转换为黑白线稿
- **可调参数**:
  - 线条深度（控制线条深浅）
  - 简化程度（控制细节保留）

#### 步骤3: 矢量优化
- **功能**: 细化和优化线条质量
- **特点**:
  - 自动去除噪点
  - 平滑线条
  - 抗锯齿处理

### 3. 实时预览对比

左右对比显示：
- **左侧**: 原始彩色图片
- **右侧**: 生成的黑白线稿

### 4. 处理结果展示

三个处理阶段的详细展示：
- 边缘检测结果
- 黑白简化结果
- 矢量优化结果

每个阶段都可独立下载

### 5. 导出功能

- 一键下载线稿图
- 支持PNG格式
- 高清输出，可直接用于纹样设计

## 🔧 技术实现

### 核心算法

#### 1. Sobel边缘检测
```typescript
// 使用3x3卷积核进行边缘检测
const sobelX = [-1, 0, 1, -2, 0, 2, -1, 0, 1];
const sobelY = [-1, -2, -1, 0, 0, 0, 1, 2, 1];
// 计算梯度幅值
edge = sqrt(gx² + gy²)
```

#### 2. 高斯模糊
- 减少噪声
- 平滑边缘
- 可调模糊半径

#### 3. 二值化处理
- 阈值分割
- 黑白转换
- 对比度增强

#### 4. 线条细化
- 去除内部填充
- 保留边缘轮廓
- 邻域分析

### 处理流程

```
原始图片
  ↓
灰度转换
  ↓
高斯模糊（可选）
  ↓
Sobel边缘检测
  ↓
阈值化
  ↓
黑白简化
  ↓
线条细化
  ↓
矢量优化
  ↓
输出线稿
```

## 📊 参数说明

### 边缘检测灵敏度
- **范围**: 0-100%
- **默认**: 50%
- **效果**:
  - 低值：检测更多细节，线条更密集
  - 高值：只保留主要轮廓，线条简洁

### 边缘平滑度
- **范围**: 0-5级
- **默认**: 1级
- **效果**:
  - 0：无平滑，保留所有细节
  - 5：高度平滑，线条流畅

### 线条深度
- **范围**: 0-100%
- **默认**: 50%
- **效果**:
  - 低值：线条浅，细腻
  - 高值：线条深，醒目

### 简化程度
- **范围**: 0-100%
- **默认**: 50%
- **效果**:
  - 低值：保留更多细节
  - 高值：简化细节，更抽象

## 📝 使用指南

### 基础使用

1. **上传图片**
   - 点击"选择文件"按钮
   - 选择要处理的图片
   - 等待图片加载完成

2. **调整参数**
   - 根据图片特点调整4个参数滑块
   - 实时查看参数说明
   - 找到最佳效果组合

3. **开始处理**
   - 点击"开始处理"按钮
   - 等待约3-5秒完成处理
   - 查看生成的线稿

4. **下载结果**
   - 点击"下载"按钮保存线稿
   - 可选择下载不同处理阶段的图片
   - PNG格式，高清无损

## ⚡ 性能优化

### 处理速度
- 小图片（<1MB）：1-2秒
- 中图片（1-3MB）：2-4秒

- 避免超大尺寸图片（>4000px）
- 图片清晰度越高，效果越好
