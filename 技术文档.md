# 染纹创合 - 完整技术文档

## 一、项目概述

### 1.1 项目简介

**项目名称**: 染纹创合（Dye Pattern Creative Hub）

**项目类型**: Web端创意设计工具

**主要用途**: 为手工艺从业者提供一个综合性的扎染纹样设计、创新、应用和管理平台

**核心目标**: 
- 通过AI技术和图像处理算法，辅助用户创新设计扎染纹样
- 支持纹样在服饰/首饰等产品上的实时预览
- 建立完整的设计历史和成果管理系统

### 1.2 项目使用语言与端口

- **应用语言**: 简体中文
- **部署方式**: Web应用（SPA），通过Netlify持续部署
- **目标浏览器**: Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- **目标分辨率**: 1920x1080（最佳体验）、最低768x480

## 三、核心功能设计与实现

### 3.1 纹样设计页面（P-PATTERN_DESIGN）

#### 功能目标
基于用户上传的参考图片，通过AI辅助和智能图像处理，动态生成创新的扎染纹样。

#### 三大核心算法

##### 1. 颜色特征提取系统

**算法**: K-means聚类算法

**实现流程**:
```
上传图片
    ↓
使用Canvas API提取像素数据
    ↓
K-means++初始化（避免局部最优）
    ↓
10次迭代优化聚类
    ↓
提取6个主要颜色簇
    ↓
计算每个颜色的占比百分比
    ↓
进行智能颜色命名（红、蓝、橙、绿等12种颜色库）
    ↓
返回完整颜色特征集合
```

**输出数据结构**:
```typescript
interface ColorInfo {
  rgb: { r: number; g: number; b: number };  // RGB值
  hex: string;                                  // 十六进制颜色码
  percentage: number;                           // 占比百分比
  name: string;                                 // 智能颜色名称
}
```

**创新点**:
- 相比传统的简单频率统计，K-means能够更精准地识别色块
- K-means++初始化避免了算法陷入局部最优
- 智能颜色命名系统使结果更易理解
- 占比计算准确反映各颜色在图片中的重要程度

##### 2. 形状特征检测系统

**算法组合**: Sobel边缘检测 + 对称性分析 + 密度计算

**子算法详述**:

**2.1 Sobel边缘检测**
```
灰度化处理
    ↓
应用Sobel 3x3卷积核
    ├── 水平梯度 (Gx)
    └── 垂直梯度 (Gy)
    ↓
计算梯度幅值 = √(Gx² + Gy²)
    ↓
计算边缘密度 = 边缘像素数 / 总像素数 × 100%
```

**2.2 对称性检测**
```
沿中心轴线进行像素镜像对比
    ↓
计算左右镜像像素的差异度
    ↓
对称性 = (相同像素数 / 总像素数) × 100%
```

**2.3 图案密度计算**
```
计算非白色/非空白像素占比
    ↓
密度 = 内容像素数 / 总像素数 × 100%
```

**输出数据结构**:
```typescript
interface ShapeInfo {
  complexity: number;    // 复杂度（基于边缘密度）0-100
  edges: number;         // 边缘密度 0-100
  symmetry: number;      // 对称性 0-100
  density: number;       // 图案密度 0-100
}
```

**创新点**:
- 从之前的"无功能"到完整的形状分析系统
- Sobel边缘检测比简单的像素差分更精准
- 多维度分析（复杂度、对称性、密度）提供全面信息
- 所有结果量化为0-100%便于理解和用于算法决策

##### 3. 智能纹样生成算法

**生成流程**:
```
提取的颜色特征 + 形状特征 + 用户参数
    ↓
决定生成策略
    ├─ 如果对称性>70% → 生成镜像对称图案
    ├─ 如果对称性<50% → 随机分布图案
    └─ 否则 → 混合策略
    ↓
根据复杂度调整纹样精细度
    ├─ 高复杂度 → 更大更精细的元素
    └─ 低复杂度 → 简单清晰的元素
    ↓
应用密度参数调整元素数量
    ├─ 高密度 → 更多纹样元素
    └─ 低密度 → 稀疏分布
    ↓
在Canvas上绘制（多个径向渐变圆形）
    ├─ 优先使用提取的主要颜色
    ├─ 保持颜色比例关系
    └─ 添加细节纹理（边缘密度>30%）
    ↓
应用用户参数
    ├─ 晕染浓度 → 控制透明度 (0-100%)
    ├─ 花纹疏密 → 控制元素数量 (0-100%)
    ├─ 融合强度 → 添加multiply混合效果 (0-100%)
    ├─ 色彩饱和度 → 调整颜色鲜艳度 (0-100%)
    └─ 纹样大小 → 控制元素尺寸 (0-100%)
    ↓
生成完整纹样并返回
```

**用户参数配置**:
```typescript
interface PatternParameter {
  dyeConcentration: number;    // 晕染浓度 0-100
  patternDensity: number;      // 花纹疏密 0-100
  fusionStrength: number;      // 融合强度 0-100
  colorSaturation: number;     // 色彩饱和度 0-100
  patternSize: number;         // 纹样大小 0-100
}
```

**创新点**:
- 基于多个图片特征的智能决策，而非简单随机
- 参数与图片特征的结合，确保生成结果更符合原图特征
- 每次点击"生成"按钮产生3个不同变体，提供选择空间
- 实时预览使用户能立即看到参数变化的效果

#### 交互流程
```
1. 用户上传参考纹样图片
   ↓
2. 系统自动提取颜色和形状特征，显示在特征面板
   ↓
3. 用户选择风格（可多选）
   ↓
4. 用户调整5个参数滑块，实时预览效果
   ↓
5. 点击"生成纹样"按钮
   ↓
6. 系统基于特征和参数生成3个变体纹样
   ↓
7. 用户可预览、下载、删除或设为"待用纹样"
```

#### 特色功能
- ✅ **双参考纹样支持**: 用户可上传两张图片，系统融合分析
- ✅ **特征可视化**: 6个颜色色块 + 4个形状指标进度条
- ✅ **实时效果预览**: 调节参数立即更新预览
- ✅ **AI模型选择**: 支持多种生成模型选择

### 3.2 素材处理页面（P-PATTERN_VECTORIZE）

#### 功能目标
将用户上传的彩色图片或扎染素材转换为黑白矢量线稿，便于后续设计使用。

#### 核心算法流程

**五步处理工作流**:

##### 步骤1-2: 预处理 → 边缘检测

```
原始图片
    ↓
灰度转换 (保留亮度信息)
    ↓
高斯模糊 (降噪，可调模糊半径 0-5)
    ↓
Sobel边缘检测 (灵敏度阈值 0-100%)
    ├─ 计算梯度幅值
    └─ 梯度 > 阈值 → 标记为边缘
    ↓
边缘二值化图像
```

**创新点**: 高斯模糊的动态可调使系统能适应不同清晰度的输入图片

##### 步骤3: 对比度增强 + 二值化

```
灰度图像
    ↓
计算图像直方图均衡化
    ↓
应用对比度增强 (参数: 0-100%)
    ↓
Otsu自适应阈值二值化
    ├─ 如果简化程度 > 50% → 更高的阈值
    └─ 如果简化程度 ≤ 50% → 更低的阈值
    ↓
得到清晰的黑白图像
```

#### 步骤4: 矢量优化

```
黑白二值图像
    ↓
线条细化算法 (Zhang-Suen或简化版)
    ├─ 删除冗余像素
    └─ 保留图案骨架
    ↓
邻域分析去噪
    ├─ 孤立点清除
    └─ 小碎片消除
    ↓
抗锯齿平滑处理
    ↓
最终矢量线稿
```

**创新点**: 
- 真正的矢量化处理，不仅仅是二值化
- 线条细化算法保持笔画的连贯性
- 多参数可调使结果适应不同使用场景

#### 用户参数配置

```typescript
interface VectorizeOptions {
  edgeThreshold: number;      // 边缘检测灵敏度 0-100%
  blurRadius: number;         // 边缘平滑度 0-5级
  contrastLevel: number;      // 线条深度 0-100%
  simplificationLevel: number;// 简化程度 0-100%
}
```

#### 处理结果

系统提供**三阶段结果展示和独立下载**:
1. **边缘检测阶段** - 显示检测到的所有边缘
2. **二值化阶段** - 纯黑白的中间结果
3. **矢量优化阶段** - 最终的清晰线稿

#### 交互流程
```
1. 上传素材图片
   ↓
2. 系统自动进行五步处理
   ↓
3. 用户在预览区看到原图 vs 线稿对比
   ↓
4. 用户可调整4个参数，实时更新预览
   ↓
5. 用户下载最终线稿或某个阶段的结果
   ↓
6. 快速跳转到纹样设计页继续创作
```

### 3.3 元素组合页面（P-ELEMENT_COMBINE）

#### 功能目标

#### 六种构图模式

##### 模式1: 对称布局
```
+-------+-------+
|   A   |   A   |
+-------+-------+
|   A   |   A   |
+-------+-------+

特点: 4个角的对称排列
应用: 适合对称艺术设计
```

##### 模式2: 置于四角
```
+-------+-------+
| A       A |
|           |
|           |
| A       A |
+-------+-------+

特点: 4个角的四散排列
应用: 框架式设计
```

##### 模式3: 中线对称
```
+-------+-------+
| A   | A |
| A   | A |
+-------+-------+

特点: 沿中线左右镜像
应用: 传统对称纹样
```

##### 模式4: 重复排列
```
+-------+-------+
| A   A   A |
| A   A   A |
| A   A   A |
+-------+-------+

特点: 网格状平铺重复
应用: 连续纹样、壁纸

##### 模式5: 放射状
```
    A
  A   A
A   C   A
  A   A

特点: 围绕中心辐射

##### 模式6: 随机分布

随机位置、随机大小、随机旋转

### 3.4 服饰应用页面（P-APPLICATION）

#### 功能目标
将设计好的纹样应用到用户上传的服饰、首饰或其他产品图片上，生成效果预览图。

#### 纹样应用算法

```
用户选择的待用纹样 + 用户上传的产品图片
    ↓
加载两张图片到Canvas
    ↓
获取纹样图片的像素数据
    ↓
设置混合模式 (globalCompositeOperation = 'multiply')
    ├─ multiply模式保持原产品色彩特征
    ├─ 纹样作为"纹理叠加"而非覆盖
    └─ 效果自然逼真
    ↓
绘制产品底图
    ↓
以乘法混合模式绘制纹样
    ↓
调整纹样尺寸和位置以贴合产品
    ↓
生成最终效果图
```

#### 混合模式选择

**为什么使用 multiply 模式?**

| 混合模式 | 效果 | 适用场景 |
|---------|------|---------|
| normal | 完全覆盖 | 不适合贴合效果 |
| multiply | 乘法混合，保留底色 | ✅ 扎染效果最逼真 |
| overlay | 强对比 | 过于刺眼 |
| screen | 叠加亮度 | 不适合深色纹样 |

**Multiply混合模式公式**:
```
结果颜色 = (纹样颜色 × 底图颜色) / 255

特点:
- 保持底图的明暗和色彩特征
- 纹样与底图融合，不显得生硬
- 深色纹样效果最佳（符合扎染特性）
```

#### 产品尺寸支持

系统支持多种输出尺寸:
```
小 (800x800)
中 (1024x1024)
大 (1440x1440)
超大 (1920x1920)
```

不同尺寸适用于:
- **小**: 网页预览、社交媒体
- **中**: 标准电商展示
- **大**: 高质量打印、详细展示
- **超大**: 4K显示、专业打印

#### 交互流程
```
1. 用户上传服饰/首饰/产品图片
   ↓
2. 从纹样管理区选择"待用纹样"
   ↓
3. 系统自动生成效果图预览
   ↓
4. 用户可调整纹样位置、大小
   ↓
5. 选择输出尺寸
   ↓
6. 点击"下载效果图"保存结果
```

#### 创新点
- Multiply混合模式使效果自然逼真
- 支持多尺寸输出适应不同需求
- 实时预览便于调整
- 一键下载便捷高效

#### 功能目标

```typescript
interface DesignHistory {

  // 纹样设计历史
  referenceImages?: {
    image1: string;
    image2: string;
  };
  parameters?: PatternParameter;
  generatedPatterns?: string[];
  appliedProduct?: string;
  
  // 其他类型数据...
}
```

## 五、AI与算法体系

### 5.1 使用的AI接口

#### 当前阶段：**无调用外部AI接口**

所有AI生成功能均通过**本地算法实现**：
- ✅ **颜色特征提取**: K-means聚类（本地）
- ✅ **形状特征检测**: Sobel边缘检测（本地）
- ✅ **纹样生成**: 参数化Canvas绘制（本地）
- ✅ **元素组合**: 几何排列算法（本地）
- ✅ **素材矢量化**: 边缘检测 + 二值化（本地）

#### 优势
- 🚀 **响应速度快**: 秒级生成，无网络延迟
- 🔒 **隐私保护**: 用户数据不上传到云端
- 💰 **成本低**: 无API调用费用
- 📱 **离线可用**: 无需网络连接

#### 未来方向
如需更高级的AI功能，可考虑接入：
- **OpenAI API**: 用于智能提示词生成
- **Hugging Face模型**: 用于更高级的图像理解
- **专有深度学习模型**: 针对扎染纹样的特殊优化

## 总结

**染纹创合**是一个集**图像处理、算法应用、AI辅助**于一身的创意设计平台。通过深度整合Canvas API、多种图像处理算法和用户友好的交互设计，为手工艺从业者提供了一个"所见即所得"的设计环境。

- 🚀 **速度**: 秒级生成，实时反馈

- 🎨 **创意**: 特征驱动，多样化输出
- 📱 **易用**: 零学习曲线，直观操作

这个项目展示了如何在**无需调用昂贵外部AI服务**的情况下，通过精心设计和优化**本地算法**来实现专业级的功能，并且提供了一套完整的**从特征提取→智能生成→参数调优→效果预览→历史管理**的工作流。
