## 🚀 立即部署步骤

### 快速部署 (推荐)
1. **创建GitHub仓库**:
   - 访问: https://github.com/new
   - 仓库名: `tie-dye-design` (或您喜欢的英文名)
   - 设置为公开仓库
   - 不要添加README、.gitignore、license

2. **运行部署脚本**:
   - 双击 `github-pages部署.bat`
   - 按提示输入您的GitHub仓库URL
   - 脚本会自动推送代码

3. **启用GitHub Pages**:
   - 进入GitHub仓库 → Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: main
   - 点击Save

### 手动部署 (如果脚本有问题)
```bash
# 提交更改
git add .
git commit -m "feat: GitHub Pages部署"

# 重新构建
npm run build

# 推送到GitHub (替换为您的仓库URL)
git remote add origin https://github.com/您的用户名/仓库名.git
git push -u origin main
```
